## User Stories

### Phase 1: MVP

#### Create account page:
As a user, I want to be able to create an account for FB-SEI3
1.	The landing page for a user that’s not logged in is a “Create Account” page.
2.	The user enters:
-	First Name
-	Last Name
-	Email or phone number (this is the username)
-	Password & password confirmation
-	Birthday
-	Gender. M, F or Other. If Other, they have to pick a pronoun (he, she, they)
3.	I have to click on the Sign-Up button at the bottom left.

#### Home page:
As a user, I want to be able to see my posts and my friends’ posts on the home page. 

#### Posting:
As a user, I want to be able to make posts. These posts can be just plain text or a photo URL.  
1.	The text field does not have a character limit.
2.	The photo URL does not have a character limit.
3.	I can like the post

#### Commenting:
As a user, I want to comment and like posts.
1.	I can enter unlimited comments on posts
2.	I can like a post once only

#### Profile page:
As a user, I want to be able to see my info and edit it.
1.	All fields entered on the sign-up page are displayed on the profile page and I can edit them if I want to

#### Milestones
1.	Set up repo on GitHub
2.	Set up Trello boards
3.	Set up Django by EOD Wednesday
4.	Have a sign-up page by noon on Thursday
5.	Base profile and home page by EOD Thursday
6.	Posts, comments, likes by EOD Friday.

### Phase 2: Beyond MVP
1.	Allow users to search posts and comments, 
2.	Apply accessibility principles to the app
3.	Use PayPal or Stripe APIs to allow people to donate to our site.
4.	Schedule when a post goes live on the platform.
5.	Send users an email when they are mentioned on the platform
6.	Integrate the chat function in the app.
